## **Urban Link 2 App**
---
##### Urban Link 2 is a modern Smart-city mobile application designed to connect users with urban services,transport systems, businesses and a real-time city information
##### .With Just a click of a button Jobs are highlighted at you finger tips.

## **Features**
---

### Some of the features are :

--**View Jobs**  View Posted Jobs

--**Save Jobs Screen** Save or even apply for a Job





## Installation
---

#### 1. Clone the Repository
````bash
git clone
https://github.com/Huntrix156/UrbanLink2
````
----
2.Open Android Studio 

3.Click **Open Project**

4.Select the UrbanLink2 folder

## Run the App
---

Shift + F10
